new - create new file (no args)
newdir - create new dir (no args)
list - list all files and dirs in dir (no args)
copyto >> <destination> - copy contents of file to existing file (1 arg)
moveto >> <destination> - move file to existing directory (1 arg)
rename >> <new name> - rename existing file or dir (1 arg)
meta - metadata of file or directory (no args)
edit >> <editor> - open file or directory in preferred editor (1 arg)
runcmd >> <cmd command> - run custom command in new terminal (1 arg)
currpath - current path (no args)
remove - remove existing file or directory (no args)
